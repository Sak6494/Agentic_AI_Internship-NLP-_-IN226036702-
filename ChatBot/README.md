# AI Chatbot with Hugging Face Transformers

A **console-based AI chatbot** built using Hugging Face pre-trained transformer models. The chatbot can understand natural language, maintain context, and generate meaningful responses dynamically.  

---

## Features
- Loads a pre-trained transformer model (GPT-2, DialoGPT, or Qwen)  
- Accepts user input from the console  
- Generates responses in real-time  
- Maintains conversation history for contextual replies  
- Exit chatbot by typing `exit` or `quit`  

---

## Requirements

- PyTorch  
- Hugging Face Transformers  
- Jupyter Notebook or VS Code
