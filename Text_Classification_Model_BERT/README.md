#  BERT-Based Sentiment Analysis (Twitter Dataset)

##  Overview

This project fine-tunes a pre-trained **BERT model** to classify Twitter text into:

* Negative
* Neutral
* Positive

---

##  Tech Stack

* Python
* PyTorch
* Hugging Face Transformers
* Scikit-learn

---

## 🚀 Workflow

* Data Cleaning
* Tokenization using `bert-base-uncased`
* Model: BERT with classification head
* Fine-tuning using AdamW (lr = 2e-5)
* Evaluation using accuracy, precision, recall, F1-score
